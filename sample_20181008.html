한국과학기술원에서 제공하는 윈도우 자동패치 서비스이용 안내입니다.
에이전트 설치가 완료되면 1~2분 후 이용 가능합니다.

          
문의처
PMS담당자
내용
귀하의 단말은 장시간 동안 KAIST PMS 서버와 통신하지 않았습니다.
에이전트를 설치 후 인터넷을 사용해 주시기 바랍니다.

O 임시 사용 : 버튼 클릭시, 7일간 인터넷 사용 후 agent 설치 안내가 뜸.

O 예외처리 대상
- MAC/Linux 기반의 VM 윈도우 사용자

O 유의사항
- 윈도우 정기 업데이트 주기 : 매월 둘째주 화, 수요일
- 윈도우 비정기 업데이트 : 윈도우 자체 상시 업데이트/ 내 PC보안점검 실행 전
- Agent가 설치 된 후, 미패치된 윈도우 보안 업데이트가 진행되므로 자신의 PC
상태에 따라 보안업데이트가 완료될때까지 기다려주시기 바랍니다.(보통 2~5분)
- 비정상적인 경로로 agent삭제시, 시스템에 이상이 발생 할 수 있으니 반드시 관리자에게 예외처리 신청
